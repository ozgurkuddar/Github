⚡ Kurulum Adımları (Cursor AI ile)
1
BotFather'dan Telegram botu oluştur
Telegram'da @BotFather'a git → /newbot → bot adı ver → token al
2
Chat ID'ni öğren
@userinfobot'a mesaj gönder veya botu başlat, /start yaz → @getidsbot chat ID verir
3
Perplexity API key al
perplexity.ai/settings/api → API key oluştur (~$5/ay)
4
.env dosyasını doldur
.env.example'ı kopyala → .env yap → 3 değeri doldur
5
Bağımlılıkları yükle ve başlat
Terminal: npm install → npm start → Telegram'dan /start gönder
6
Railway'e deploy et (opsiyonel)
railway.app → New Project → GitHub repo bağla → env değişkenlerini ekle → Deploy
7
Bu paneli GitHub Pages'e yükle
GitHub repo → Settings → Pages → index.html'i root'a koy → Deploy
